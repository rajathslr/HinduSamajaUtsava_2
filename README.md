# Hindu Samajotsava - Varahasandra

A beautiful, modern website for the Hindu Samajotsava event at Basaveshwara Temple, Varahasandra, Sompura.

## Event Details

- **Date**: 18-01-2026, Sunday
- **Time**: 10:30 AM Onwards
- **Venue**: Basaveshwara Temple, Varahasandra, Sompura

## Features

- **Modern Design**: Purple/violet color scheme with golden accents
- **Responsive Layout**: Works perfectly on all devices
- **SEO Optimized**: Meta tags, sitemap, and robots.txt included
- **QR Code**: Automatically generated QR code for easy sharing
- **Smooth Animations**: Beautiful hover effects and scroll animations
- **Accessibility**: Follows web accessibility best practices

## Design Highlights

This website uses a **purple and violet color palette** to differentiate it from the original Hindu Samajotsava event website (which uses saffron/orange colors). Key visual elements include:

- Purple gradient backgrounds (#7B2CBF to #9D4EDD)
- Golden accents (#FFD700) for emphasis
- Smooth transitions and hover effects
- 3D pop-out effects on cards
- Animated Om symbols with glowing effects

## Technologies Used

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with CSS variables, gradients, and animations
- **JavaScript**: Interactive features and QR code generation
- **Google Fonts**: Cinzel (headings), Poppins (body), Noto Sans Devanagari (Om symbols)
- **QRCode.js**: For generating QR codes

## How to Run

1. Simply open `index.html` in any modern web browser
2. For local development with live reload, you can use:
   - Python: `python -m http.server 8000`
   - Node.js: `npx serve`
   - VS Code: Use Live Server extension
