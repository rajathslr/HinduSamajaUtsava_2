# Event Comparison: Hindu Samajotsava

This document highlights the key differences between the two Hindu Samajotsava events.

## Visual Differences

### Original Event (HinduSamaj)
- **Color Scheme**: Saffron, Orange, Crimson, and Golden
- **Primary Colors**: #FF6B35 (Orange), #D2691E (Saffron)
- **Theme**: Warm, traditional saffron theme

### New Event (HinduSamaj_2)
- **Color Scheme**: Purple, Violet, and Golden
- **Primary Colors**: #7B2CBF (Purple), #9D4EDD (Violet), #C77DFF (Light Purple)
- **Theme**: Cool, regal purple theme with golden accents

## Program Details

Both events follow the same program structure:
1. A grand procession
2. Gopuja
3. Lighting the lamp & Bharath Mata Puja
4. Blessings from Swamiji
5. Addresses by honorable guests
6. Cultural Program
7. Singing of Vande Mataram by all
8. Prasada Distribution

## Purpose of Differentiation

The distinct color schemes help:
1. **Visual Identity**: Each event has its own unique brand identity
2. **Easy Recognition**: Attendees can quickly identify which event materials belong to which location
3. **Professional Appearance**: Both events maintain high-quality, premium aesthetics
4. **Cultural Respect**: Both honor Hindu traditions while presenting modern, accessible designs
